# Threat Intelligence Aggregator

## Overview
This project focuses on aggregating and analyzing threat intelligence data for defensive cybersecurity use cases.
It was built as part of my learning in blue-team security and SOC operations.

## Objective
- Collect and normalize threat intelligence indicators
- Support security monitoring and analysis
- Understand how threat intelligence feeds are used in practice

## Scope
- Defensive security research only
- No exploitation or offensive activity
- No real-time attack execution

## Tools & Technologies
- Python
- Public threat intelligence sources
- Linux environment

## Ethical Notice
This project is intended strictly for educational and defensive purposes.
It does not perform exploitation, scanning, or unauthorized activity.
